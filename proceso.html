¿Cómo funciona la atención médica virtual?

Nuestro proceso está diseñado para brindar una atención segura, ordenada y centrada en el paciente.

1️⃣ Solicitud de cita

El paciente solicita una consulta mediante formulario digital.

2️⃣ Formulario digital

Se recopilan datos básicos del paciente, motivo de consulta y antecedentes importantes.

3️⃣ Validación de datos

El equipo médico revisa la información proporcionada.

4️⃣ Asignación de fecha y hora

Se programa la consulta virtual según disponibilidad.

5️⃣ Videollamada médica (Sincrónica)

Comunicación en tiempo real entre médico y paciente para evaluación y orientación médica.

6️⃣ Evaluación y orientación médica

El médico analiza la información, explica recomendaciones y planifica seguimiento.

7️⃣ Envío de indicaciones digitales

Se entregan recomendaciones, documentos o indicaciones médicas.

8️⃣ Seguimiento por mensajería (Asincrónico)

Seguimiento mediante:
✓ Mensajes
✓ Envío de resultados
✓ Revisión de documentos

9️⃣ Alta médica o referencia presencial

Si el paciente requiere evaluación física, se realiza referencia a atención presencial.
